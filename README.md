## What is InflationIQ?
 
InflationIQ turns complex macroeconomic data into plain-English forecasts — instantly. Pick a country, pick a commodity, and get a real AI-generated outlook complete with risk narrative, business impact, and a concrete investment signal. No economics degree required.

## Features
 
- **🌍 8 Major Economies** — US, EU, UK, Japan, Brazil, Turkey, and more

- **🤖 Claude 3.5 Sonnet** — Every forecast is reasoned by Anthropic's Claude in real time

## Getting Started
 
### Prerequisites
- Node.js 18+ or Bun 1.x
- An Anthropic API key → [console.anthropic.com](https://console.anthropic.com)
### Local Development
 
```bash
# Clone the repo
git clone https://github.com/k-kj0/inflation-insight-ai
cd inflation-insight-ai
 
# Install dependencies
npm install
 
# Add your API key
echo "VITE_ANTHROPIC_API_KEY=your_key_here" > .env
 
# Start dev server
npm run dev
```
 
Open [http://localhost:5173](http://localhost:5173) in your browser.
 
### Build for Production
 
```bash
npm run build
# Output is in /dist
```

## How It Works
 
1. **User picks** a country + commodity from the predictor UI
2. **InflationIQ builds** a structured economic prompt with current macro context
3. **Claude 3.5 Sonnet** reasons over the data and returns a forecast, risk narrative, business impact summary, and investment signal
4. **Results render** in under 3 seconds directly in the browser
